Add-Ons

Interactive Content

  • Astro, a collection of mobile-first navigation patterns.
  • Drop, mobile-friendly dropdown menus.
  • Houdini, a simple expand-and-collapse widget.
  • Tabby, lightweight, mobile-first toggle tabs.
  • Jellyfish, lazy load images and iframes.
  • Modals, simple modal dialogue windows.
  • X-Ray, let's users toggle password visibility in forms.
  • Form Saver, a lightweight script for saving form data.
  • Slider, a responsive, touch-enabled content slider.
  • Smooth Scroll, a simple script to animate scrolling to anchor links.
  • Right Height, variable content with equal heights.
  • Sticky Footer, dynamic, responsive sticky footers.

CSS Components

  • Snapshot, basic image styling.
  • Tables, simple CSS table styling.
  • Hero, basic callout boxes.
  • Social Sharing links and buttons without the bloat.
  • Progress Bars, lightweight CSS progress bars.
  • Alerts, simple alert messages.
  • Labels, lightweight CSS labels.
  • Backgrounds, basic section dividers.

Other People's Stuff

  • FluidVids.js, a lightweight javascript snippet for fluid iframes.
  • Prism, a lightweight syntax highlighter by Lea Verou.
  • .htaccess, best practice server rules for making web pages fast and secure.

WordPress Basics

  • Keel, a boilerplate for WordPress theme development.
  • HTML Minify, compress your HTML output.
  • Google Analytics, add an optimized Google Analytics script.
  • Antispambot, hide your email address from spam robots.
  • Tarpit, eliminate comment spam.
  • Remove Header Junk, removes the junk WordPress adds to the header.

WordPress Specialty

  • Google-Hosted jQuery, use Google's CDN for better performance.
  • Image Compress & Sharpen, reduce image file size.
  • Exclude From Search, exclude pages and posts from search results.
  • Add IDs to Header Tags Plus, adds IDs to post headings.
  • WordPress for Web Apps, transform WordPress into a web app engine.
  • WordPress Theme Options, adjust theme settings from the admin dashboard.
  • Petfinder for WordPress, functions for working with the Petfinder API.
