Forms

Labels, legends and inputs are styled as full width block elements (with the exception of checkboxes and radio buttons).

<form>
    <label>Label</label>
    <input type="text">
    <label>
        <input type="checkbox">
        Option 1
    </label>
    <label>
        <input type="checkbox">
        Option 2
    </label>
    <label>
        <input type="radio" name="radioset">
        Option 1
    </label>
    <label>
        <input type="radio" name="radioset">
        Option 2
    </label>
    <select>
        <option>Select 1</option>
        <option>Select 2</option>
        <option>Select 3</option>
    </select>
    <textarea></textarea>
</form>

Pro Tip: Wrap checkboxes and radio buttons inside a <label></label> to make them easier to click.

Form Layouts

Use the grid system to add structure to your forms.

<form>
    <div class="row">
        <div class="grid-2">
            <label>Name</label>
        </div>
        <div class="grid-4">
            <input type="text">
        </div>
    </div>
    <div class="row">
        <div class="grid-2">
            <label>Message</label>
        </div>
        <div class="grid-4">
            <textarea></textarea>
        </div>
    </div>
    <div class="row">
        <div class="grid-4 offset-2">
            <button class="btn btn-blue">Submit</button>
        </div>
    </div>
</form>

New: Condensed & Inline Inputs

For smaller forms, add the .input-condensed class to your input fields. For inline form elements, add the .input-inline class.

<form>
    <input type="text" value=".input-inline" class="input-inline">
    <input type="text" value=".input-condensed" class="input-condensed input-inline">
</form>
